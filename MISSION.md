# Mission: World Cup 2026 ML/DL Predictor

## Why
Build a portfolio-worthy capstone project that predicts every match of the 2026 FIFA World Cup and simulates the tournament to predict the winner — demonstrating end-to-end ML/DL skills (data engineering, feature engineering, model building, evaluation, deployment) to a hiring manager or graduate admissions committee.

## Success looks like
- A trained ensemble of ML and DL models that predict match outcomes (win/draw/loss + scoreline) with accuracy meaningfully above random (>45% on 3-class)
- A Monte Carlo tournament simulator that predicts the winner and bracket from match-level probabilities
- Backtested predictions against the 2022 World Cup with documented performance metrics
- Real-time predictions for the ongoing 2026 World Cup that can be compared against actual results
- Polished Jupyter notebooks that walk through the entire pipeline (EDA → features → models → evaluation)
- An interactive Streamlit dashboard presenting predictions, bracket, and model insights
- A portfolio-quality README with methodology, results, and a live demo link

## Constraints
- New to ML/DL — needs to learn-by-doing with clear guidance at each step
- Time-sensitive: 2026 World Cup is live right now (June 11 – July 19, 2026)
- No paid APIs — use free Kaggle datasets and publicly available data
- Python ecosystem (pandas, scikit-learn, XGBoost, PyTorch, Streamlit)

## Out of scope
- Real-time scraping of live match events (use pre-match features only)
- Betting/gambling applications
- Player-level injury tracking or transfer market data
- Mobile app development
